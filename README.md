# Movie-app

An application built to fetch movie data using the OMDb API. It features movie search functionality, detailed movie information views, and a watchlist manager. It includes a custom StarRating component for user ratings. The app is powered by useState and useEffect hooks, along with three custom hooks. I also used local storage to save movies to the list.

Aplikacija napravljena da fatch-uje podatke filmova koristeci OMDb API. Moguca je pretraga filmova,pregled podataka o filmu, dodavanje u listu gledanih, i napravljena je komponenta StarRating pomocu koje mozemo dodati nasu ocenu filma. Koriscene su useState i useEffect hook-s a takodje je napravljeno i tri custom hooks-a. Takodje sam koristio i local-storage da sacuvam filmove na listi.
